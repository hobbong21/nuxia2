## Grafana Dashboard (docs/grafana-dashboard.json)

5 패널 (Grafana 10+ schemaVersion 39):
1. **Referral Distribute Rate** (timeseries) — `sum by (result) (rate(nuxia2_referral_distribute_total[5m]))`
2. **Payment Confirm Rate by Result** (stacked timeseries, mismatch 강조 색상) — `sum by (result) (rate(nuxia2_payment_confirm_total[5m]))`
3. **Abuse Blocked by Kind** (bar gauge, 1h increase) — `sum by (kind) (increase(nuxia2_abuse_blocked_total[1h]))`
4. **Webhook Received by Source/Status** (timeseries, rejected 빨강) — `sum by (source, status) (rate(nuxia2_webhook_received_total[5m]))`
5. **Minor Hold Count** (stat panel) — `nuxia2_minor_hold_total`

- time: `now-6h`, refresh 30s
- datasource 템플릿 변수 1개 (`$datasource`)
- 각 패널에 id / gridPos / targets / fieldConfig / options 최소 구조

## Prometheus Alerts (docs/prometheus-alerts.yml)

`nuxia2-alerts` 그룹, interval 30s, **4 룰**:

| Alert | severity | trigger |
|-------|----------|---------|
| PaymentMismatchSpike | critical | 5분 rate `mismatch` > 5/300, for 2m |
| WebhookRejectedSpike | warning | 5분 rate `rejected` > 3/300, for 2m |
| ReferralDistributeFailure | critical | 5분 rate `failed` > 0, for 1m |
| MinorHoldGrowth | info | 1h increase `minor_hold_total` > 10, for 5m |

모두 `team: nuxia2` 라벨 포함 → Alertmanager 라우팅 키.
